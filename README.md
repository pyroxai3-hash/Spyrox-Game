# SPYROX

SPYROX is an open-world creature survival game set on the planet Athena.

The game includes exploration, creature collection, vehicles, puzzles, and multiplayer survival.

## Planned Features

* 30 unique regions
* 1000 Spyrox creatures
* 1000 customizable Pyrox vehicles
* Open world exploration
* Multiplayer co-op survival
* Base building and crafting
